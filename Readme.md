# 💥 Harsha Vardhan Villuri | Comical Theory Portfolio

Welcome to the **"Comical Theory"** — a highly interactive, Neo-Brutalist, comic-book themed personal portfolio designed and built from scratch. 

This project completely ditches boring, generic developer templates and embraces a raw, high-energy UI/UX aesthetic heavily inspired by classic comic books, pop-art, and cyberpunk "Night City" vibes.

## ✨ Epic Features & Micro-Interactions

This portfolio isn't just a static website; it is packed with complex JavaScript interactions and web-audio synthesizers:

* **🧠 Neural Network Canvas:** A living, breathing interactive HTML5 Canvas background that draws connecting "neural" nodes to your cursor.
* **🌙 Night City Dark Mode:** Instantly shifts the vibrant comic colors into a gritty, neon-glowing Cyberpunk theme, completely swapping out the main hero avatar for a "Night City" variant and adding a flashlight spotlight effect to the cursor!
* **🎮 GOD MODE (Konami Code):** A secret keyboard event listener. Type `UP UP DOWN DOWN LEFT RIGHT LEFT RIGHT B A` to unlock God Mode, complete with a massive explosion and confetti rain.
* **💬 Comic Speech-Bubble Contact Form:** A fully custom contact form styled as speech bubbles. When submitted, the form folds up and physically flies off the screen like a paper airplane.
* **🧲 Magnetic Buttons:** A premium UI/UX touch where buttons physically pull toward your mouse cursor as you hover near them.
* **🖨️ Printing Press Loader:** A custom CSS animation that simulates an old-school printing press stamping out "Issue #1" when the site first loads.
* **📄 Secret File Resume:** A fully integrated, completely custom web-based resume (`resume.html`) styled to match the exact comic aesthetic of the main portfolio.

## 🛠️ Tech Stack (The Superpowers)

Built strictly with core web technologies to ensure maximum performance and complete custom control without relying on heavy frameworks:

* **HTML5:** Semantic, accessible structure.
* **CSS3:** Custom Neo-Brutalist variables, complex `border-radius` math for hand-drawn wobbly boxes, CSS grid/flexbox, and keyframe animations.
* **Vanilla JavaScript (ES6+):** Handling all DOM manipulation, custom cursors, Canvas drawing algorithms, and the Web Audio API for synthesized retro sound effects.
* **Web Audio API:** Generates dynamic sound effects (beeps, pows, tearing paper, glitches) completely through browser synthesis—no audio files needed!

## 🚀 How to Run Locally

You don't need any complex build tools to run this. It is pure Frontend code!

1. Clone or download this repository.
2. Open the `Task2-Portfolio` folder.
3. Double click on `index.html` to open it in your web browser.
4. Enjoy the experience! 
*(Note: To hear the synthesized audio effects, you may need to click the "Unmute" button in the top right navigation bar).*
